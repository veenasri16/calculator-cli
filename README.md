# calculator-cli
Simple Python CLI Calculator

The Calculator CLI Application is a simple command-line based calculator built using Python.
It performs basic arithmetic operations such as addition, subtraction, multiplication, and division.


This project demonstrates fundamental Python concepts including
Functions and Loops used:
* Conditional statements
* Error handling
* User input handling
 The program runs continuously until the user chooses to exit.

-Features Used :
  * Addition
  * Subtraction
  * Multiplication
  * Division
  * Division-by-zero handling
  * Continuous execution until exit
  * Input validation for non-numeric values
Technologies Used :
  -Python 3
  =Command Line Interface (CLI)
